AI Search Settings

When enabled, your search query is sent to an LLM which suggests relevant conference acronyms from its training knowledge and expands your keywords — then results are looked up in the local CORE / SCImago data. Your API key is stored only in your browser (localStorage) and never sent anywhere except the provider you choose.

Get a free Groq key at console.groq.com/keys · OpenAI: platform.openai.com/api-keys · Gemini: aistudio.google.com

Scopus proxy (optional)

To use Elsevier Scopus from this page, run the small Python server in server/scopus_proxy.py on a host you control and paste its public URL below. Your SCOPUS_API_KEY lives only on that server. The Pages site must be served over HTTPS, so the proxy URL must also be HTTPS (mixed content is blocked in browsers).

Use the service root (we call …/literature-venues) or paste the full endpoint URL ending in literature-venues.

See CLAUDE.md for SCOPUS_PROXY_ALLOW_ORIGIN and deployment notes.
